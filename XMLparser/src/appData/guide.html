User Guide

Menu bar functions

  • Files
    • Show All = Show selected files view
    • Add New = Add files from disk (use CTRL to select multiple)
    • Remove All = Remove all selected files
  • Help
    • User Guide = Show user guide view
    • Keybinds = Show keybinds view

Parser commands (Show All)

  • File schema
    • The program parses XML files with the following schema. The first line may include optional tag "?xml version='1.0'?". All other data must be enclosed inside a root node, which is named, for example, "root". Inside root node, credit and debit nodes can be placed freely and they can be placed inside month node with attribute number = String(Integer). Check src/testData files for reference.
  • Running Parser
    • After adding files form a disk to the files view, user may browse through the files and open them by pressing a button named after the file. To parse a file, user must select the corresponding radio button next to filename and then press "Run Parser"" button at the bottom. Similarly, user can delete single selected file by pressing "Delete File" button.
  • Parsing results
    • Total Balance includes three figures. First value indicates the total sum all credit and debit values in the file, whether or not they have given a month attribute. The Second value indicates the total sum of cases with a month attribute value and the third value the total sum of cases without the month attribute. Therefore the sum of the second and the third value should be equal to the first value.
    • Finally, the total sums of credit and debit values of each month are shown. An invalid month contains values outside the closed interval [1,12]. However, making the month attribute not a string presentation of integer will cause an error to the program.
